# RokiArkNet Plugins

Custom ARK: Survival Ascended server plugins for enhanced gameplay.

## Available Plugins

| Plugin | Description | License |
|--------|-------------|---------|
| [EngramControl](engramcontrol.md) | Block engrams and unlock via boss kills | Annual |
| [DinoLoot](dinoloot.md) | Add custom loot drops to dino corpses | Annual |
| [TekUnlock](tekunlock.md) | Unlock Tek engrams by killing bosses | Annual |
| [DinoLevelOverride](dinoleveloverride.md) | Override wild dino level distributions | Free |
| [RiderProtection](riderprotection.md) | Protect mounted players from damage | Free |
| [ItemExchange](itemexchange.md) | Exchange items between players | Free |

## Installation

1. Download the plugin zip from the releases page
2. Extract to `ShooterGame/Binaries/Win64/ArkApi/Plugins/`
3. Restart server or use hot-reload (`.ArkApi` extension)

## Licensing

Premium plugins require an annual license key. Purchase at [ko-fi.com/rokiarknet](https://ko-fi.com/rokiarknet).

Free plugins have no restrictions.
