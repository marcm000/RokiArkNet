# RokiArkNet

Custom ARK: Survival Ascended server plugins and mod guides.

## Plugins

Server plugins for ARK: Survival Ascended using ArkApi.

| Plugin | Description | License |
|--------|-------------|---------|
| [EngramControl](plugins/engramcontrol.md) | Block engrams and unlock via boss kills | Annual |
| [DinoLoot](plugins/dinoloot.md) | Add custom loot drops to dino corpses | Annual |
| [TekUnlock](plugins/tekunlock.md) | Unlock Tek engrams by killing bosses | Annual |
| [DinoLevelOverride](plugins/dinoleveloverride.md) | Override wild dino level distributions | Free |
| [RiderProtection](plugins/riderprotection.md) | Protect mounted players from damage | Free |
| [ItemExchange](plugins/itemexchange.md) | Exchange items between players | Free |

[View All Plugins](plugins/index.md){ .md-button }
